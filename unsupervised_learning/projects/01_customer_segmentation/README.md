# 🎯 项目1: 客户分群 (Customer Segmentation)

> 使用K-Means和GMM对客户进行分群分析

---

## 📋 项目概述

**业务场景**：
一家零售商希望了解客户的购买行为模式，以便制定针对性的营销策略。

**项目目标**：
1. 使用聚类算法将客户分成不同的群体
2. 分析每个客户群的特征
3. 为每个客户群提供营销建议

**技术栈**：
- K-Means聚类
- 高斯混合模型(GMM)
- 轮廓系数、肘部法则
- 数据可视化

---

## 📊 数据集

### 数据来源
- Kaggle: [Mall Customer Segmentation](https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python)
- 或使用模拟数据

### 数据字段
- **CustomerID**: 客户ID
- **Gender**: 性别
- **Age**: 年龄
- **Annual Income (k$)**: 年收入（千美元）
- **Spending Score (1-100)**: 消费评分（1-100）
