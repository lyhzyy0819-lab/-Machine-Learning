# 📂 无监督学习实战项目

> 3个实战项目，帮助你巩固无监督学习知识

---

## 🎯 项目列表

### 项目1: 客户分群 (Customer Segmentation)
**难度**: ⭐⭐
**时间**: 2天
**技术**: K-Means, GMM, 轮廓系数

**目标**：
- 使用聚类算法对客户进行分群
- 分析每个客户群的特征
- 提供营销建议

**数据集**：
- Mall Customer Segmentation (Kaggle)
- 或使用模拟数据

**deliverables**：
- 聚类分析Notebook
- 客户画像报告
- 可视化结果

---

### 项目2: PCA图像压缩 (Image Compression with PCA)
**难度**: ⭐⭐⭐
**时间**: 1.5天
**技术**: PCA, 方差解释率

**目标**：
- 使用PCA压缩图像
- 分析不同压缩率的效果
- 计算压缩比和重构误差

**数据集**：
- Olivetti Faces
- 或自定义图像

**deliverables**：
- PCA图像压缩Notebook
- 压缩率 vs 质量曲线
- 重构图像对比

---

### 项目3: 欺诈检测 (Fraud Detection)
**难度**: ⭐⭐⭐
**时间**: 1.5天
**技术**: Isolation Forest, One-Class SVM, 不平衡数据处理

**目标**：
- 使用异常检测算法识别欺诈交易
- 处理高度不平衡数据
- 评估模型性能

**数据集**：
- Credit Card Fraud Detection (Kaggle)

**deliverables**：
- 欺诈检测Notebook
- 模型评估报告
- ROC曲线和混淆矩阵

## 🚀 开始项目

### 项目工作流

1. **理解业务问题**
   - 明确项目目标
   - 了解数据背景
   - 定义成功标准

2. **探索性数据分析 (EDA)**
   - 查看数据基本信息
   - 可视化特征分布
   - 检测缺失值和异常值

3. **数据预处理**
   - 处理缺失值
   - 特征工程
   - 数据标准化

4. **模型训练与评估**
   - 选择合适的算法
   - 参数调优
   - 模型评估

5. **结果可视化与解读**
   - 可视化结果
   - 业务解读
   - 提出建议

6. **文档与总结**
   - 写清楚思路和步骤
   - 记录遇到的问题和解决方案
